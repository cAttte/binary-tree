# 🌲 binary-tree

inverting binary trees like a boss 😎

## Installation

    $ npm install cAttte/binary-tree

## API

the module exports a single (one (1)) class, _Node_:

```js
const Node = require("binary-tree")
```

```ts
import Node from "binary-tree"
```

_Node_ has the following methods:

### constructor(value: number, left?: Node, right?: Node)

-   **value:** the value of this node (duh).
-   **left:** the left pointer of this node (duh).
-   **right:** the right pointer of this node (duh).

### clone(): Node

clone this node! useful because all of the other methods mutate their nodes.

### equals(that: Node): boolean

find out whether _this_ node equals _that_ node (including children). how handy!

### find(value: number): Node

find a value within the tree. returns `null` if not found!

### insert(value: number): this

insert a value into the tree.
